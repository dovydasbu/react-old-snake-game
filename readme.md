React based snake game. Main purpose - use as little packages as possible to grasp basic main principles of plain react.

## View
![alt text](http://i67.tinypic.com/23w16ar.jpg)

## Installation

```bash
npm install
npm run start
```

## Controls

- With arrow keys
---
- w goes &uarr;
- d goes &rarr;
- s goes &darr;
- a goes &larr;
